1. R이나 RStudio의 명령 창에서 다음 명령을 붙여넣기하고 엔터를 치면 모두 설치됩니다.

install.packages(c('rvest', 'httr', 'stringr', 'readxl', 'dplyr', 'tm', 'qgraph'), repos='http://cran.nexr.com')

2. KoNLP의 경우 먼저 JAVA를 설치 후 KoNLP 패키지를 설치하시기 바랍니다. 

윈도우의 경우 JAVA 설치법
MAC의 경우 JAVA 설치법
install.packages('KoNLP', repos='http://cran.nexr.com')

윈도에서 RStudio가 패키지 설치를 못할 경우

윈도에서 사용자 이름이 한글인 경우 RStudio에서 설치가 잘 되지 않을 수 있습니다. 이때는 다음 둘 중 한 가지 방법을 사용하세요.

  • R에서 설치하시면 잘 됩니다.
  • TEMP와 TMP 환경 변수를 한글이 없는 경로(예: C:\TEMP 등)로 설정하면 RStudio에서도 설치가 잘 됩니다
    • 환경변수는 제어판에서 사용자 계정을 선택 후 환경변수 변경을 선택하면 설정할 수 있습니다.
